## 二、关键突破（坑 → 解）
| 坑 | 解 |
|----|----|
| 旧 session-token 过期「Invalid auth token」 | 走真实登录流取新鲜 `ott$...` |
| 邮箱 `@` 输入被吞 | 剪贴板粘贴（Set-Clipboard + Ctrl+V） |
| relay 中文路径/输出丢字（隔字掉字） | `dao_b64.py` 命令+输出双向 base64；`dao_put.py` 路径+内容双 base64，SHA256 字节无损 |
| GitHub 私仓 git clone 走代理鉴权失败 | 改 GitHub API tarball 下载（Bearer token，`dl_repo.py`） |
| 141 命令队列饱和超时(408) | 退避重试 + 把只读查询切到空闲的 179 |
| relay exec-sync 串行阻塞 | 大任务用 subprocess.Popen 后台 + 轮询 |
