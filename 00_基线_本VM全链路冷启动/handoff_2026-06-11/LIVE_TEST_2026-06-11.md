### 安全边界（继承自 v2.0.0 隔离测试）
- `/api/*` 路径操作经 `withinRoot()` 限定在工作区根：合法路径 200，无 Token 401，越权路径（`..` 逃逸）403。
- 网络受限环境用 `--protocol http2` 强制 TCP，绕过本机 QUIC/UDP 封锁，Cloudflare 边缘正常路由。

## 已知限制（非 bug）

1. **dao-vsix 全功能面板拉取 Devin 数据（Sessions/Knowledge）需 `cog_` 开头的 Devin API Key。**
   当前 Free 账号经 rt-flow 注入的是 `devin-session-token$`（仅对 Codeium API 有效），面板已正确检测并提示「当前认证仅对 Codeium API 有效，Devin API 返回 403」。
   用 auth1 / `cog_` token 时本会话早前已实测拉到 **104 条真实会话** + 注入帛书规则知识库（note 200）。属账号能力差异，逻辑无误。
2. **Quick Tunnel URL 每次启动会变**（无自定义域名时的预期行为，正合「每次新通道」需求）。始终以最新 `workspace.md` 为准。
